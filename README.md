# Voidpx
Voidpx is a clean, lightweight, and responsive pixel art gallery built with HTML, CSS, and Vanilla JavaScript. It provides an intuitive way to browse, search, preview, and download pixel art assets while offering a modern retro-inspired interface. Users can filter artworks by category, save favorites locally, switch between dark and light themes, and copy color palettes with a single click.

<p align="center">
    <img src="https://i.postimg.cc/G2nc0J6h/Voidpx.png" alt="Voidpx">
</p>

## Features
- 🔍 Search by image title
- 🗂️ Category filtering
- ❤️ Favorites system with Local Storage
- 🌙 Dark & Light theme
- 🎨 Display color palettes
- ⬇️ Download images
- 📄 Responsive pagination
- 🔗 Shareable URL state (page, search, category)

## Installation
Clone the repository and open the project in your browser.<br>
```bash
git clone https://github.com/mammaddrik/Voidpx.git
cd Voidpx
```
Run the project using any local web server.
